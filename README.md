### CSS guide

Als eerst begin ik bovenaan de code met algemene styling dat geld op elk device.
```
@media (min-width: 60rem){
  ...
}
```

Ten tweede met de tablet
```
@media (min-width: 30rem) and (max-width: 60rem){
  ...
}
```
Tot slot met mobiel
```
@media (max-width: 30rem){
  ...
}
```
